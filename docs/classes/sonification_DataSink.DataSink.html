Optional onStreamEnded

onStreamEnded?: () => void
  • Defined in sonification/DataSink.ts:35

Type declaration

    • (): void

    • An event hook to know when the data stream for a DataSink has ended. The OutputEngine needs to handle this for each DataSink to know when a sink has stopped because the data stream has ended. If this event is not handled, the OutputEngine will not know when the datastream for the sink has ended. Consequently, the OutputEngine's outputState never goes to stop after initial call to onPlay.

      Returns void
